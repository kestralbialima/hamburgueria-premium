
# 🍔 Sua Hamburgueria - Sistema de Alta Performance

Sistema de cardápio digital inteligente desenvolvido para micro e pequenos empresários do ramo gastronômico.

### ✨ Funcionalidades:
- 🛒 Carrinho dinâmico e inteligente.
- 📍 Localização automática via GPS.
- 💬 Finalização de pedido via WhatsApp formatado.
- 🚀 Design Ultra-Rápido com Tailwind CSS.
- 📱 Interface 100% responsiva (Mobile First).

### 💰 Valor Agregado:
Este projeto ajuda o empresário a economizar em taxas de delivery e agilizar o atendimento no WhatsApp em até 70%.
